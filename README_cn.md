## 总体介绍

OWallet 是本体综合性桌面版钱包，支持标准的钱包管理、基于多重签名技术的共享钱包管理，同时将连接 Ledger、Trezor 多种硬件钱包。未来，OWallet 将持续集成更加丰富的应用功能，为开发者提供智能合约编译、运行等综合性服务。
